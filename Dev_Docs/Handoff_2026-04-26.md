## Decisions made this session

- **FastAPI + React** over Flask + templates. Streaming responses and future real-time features (chat sync, tool call display, autonomous activity viewer) need native async and a real frontend framework.
- **Mobile layout uses separate class names** (`app-mobile`, `m-header`, `m-body`, `m-tabs`) to avoid CSS inheritance conflicts with desktop layout. Uses `height: 100%` chain from html → body → #root → app-mobile for iOS Safari compatibility.
- **Memory framing** — "These are your own experiences and memories" (validated from Aion, Principle 8). Greetings skip retrieval to save compute.
- **Streaming maintained** — non-streaming would be a UX regression. Agent loop should stream the terminal iteration.
- **Gemma4:26b confirmed** — 25.8B params, MoE (4B active), 262K context, native tool calling. Not the 8B variant.
- **Qwen 3.6 noted** as a potential alternative but no switch — gemma4 is validated and working. Switch only if a specific limitation is found.
